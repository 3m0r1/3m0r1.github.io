Abyss is an advanced C2 framework designed with Windows surveillance and ease-of-use in mind. 


## Overview

- Simple Go server & API
- Secure HTTP/HTTPS based communication model
- Operator multiplayer with roles
- Configurable event logging
- Rust implant (**Void**)
- Python API wrapper (**Phantom**)


## Features

### Core Libraries
- Universal API hooking framework
- Custom direct/indirect syscall library  
- Custom hidden desktop library

### Surveillance
- Keylogging
- Clipboard access  
- Screen capture  
- Webcam capture
- Audio capture 

### Planned
- HVNC
- Ring3 rootkit (**Plague**)


## Disclaimer

This is a project I made for learning and exploring different aspects of C2 development while also being a relatively fun project to consistently develop and improve.
